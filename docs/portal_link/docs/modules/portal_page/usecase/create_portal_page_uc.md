# Create Portal Page

## 概述

此用例允許已登入使用者建立個人化的連結整合頁面（Portal Page）。使用者可以設定頁面的基本資訊，包括自訂網址（slug）、標題、個人簡介、頭像和主題風格。

**主要參與者：** 已登入使用者
